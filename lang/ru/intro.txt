====== Set Feedback Managers ======

Эта форма позволит Вам задать email адреса людей, которые должны получать сообщения обратной связи по конкретным пространствам имен.

Назначенные адреса используются и для вложенных пространств имен. Можно задать несколько адресов через запятую. Для определения корневого пространства имен используйте звездочку ''*''.
